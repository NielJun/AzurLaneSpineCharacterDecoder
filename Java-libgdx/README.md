## 使用方法

### 方法1
在 azurlanespine.jar 的同级目录打开cmd

输入 skel文件的完整路径（多个）或存储有全部文件的json文件

回车启动

当背景为白色，表示正在导出
当背景转为蓝色，表示完成
导出位于运行目录下的output文件夹中
